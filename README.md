## Run Locally

**Prerequisites:**  Node.js

Open the Cursor or terminal.
1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
2.1 For the free API key
   https://aistudio.google.com/apikey Setup your first time google cloud and get the 300 credit.
4. Run the app: Type this below in the terminal
   `npm run dev`

## 한국어설명서

  Node.js 가 필요합니다.

Cursor 나 VS 코드같은 editor 툴이 필요합니다. 해당 툴을 쓰면 터미널을 활성화 할수있습니다.
그게 싫으시면 윈도우 파워셸을 쓰셔도 됩니다.
1. Install dependencie:(라이브러리 설치를해야합니다)
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
   AI는 Gemini api키를써야합니다.
   첨부된 .env.local 을 수정해야합니다. (코드 에디터가 없으면 메모장으로 하셔도 됩니다.)
2.1 For the free API key
   현주 구글은 구글클라우드 홍보를 위해 300크레딧을 무료로 뿌리는데 이걸 AI 용도로쓰셔도 됩니다.
   https://aistudio.google.com/apikey Setup your first time google cloud and get the 300 credit.
4. Run the app: Type this below in the terminal
   앞선 작업이 끝났으면 터미널에서 실행하시면 됩니다.
   `npm run dev`
참고로 페르소나는 Add 를 누르면 계속 추가할수 있기때문에 50명 100명 가능합니다.
다만 질문은 10개가 현재 한계입니다.
